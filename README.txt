Full Flask House Price Project
-----------------------------
How to use:
1. Place your CSV at data/house_data.csv (columns should include: Price, living area, number of bedrooms, number of bathrooms)
2. (Optional) Run python train_model.py to train a model and produce linear_regression_model.pkl and model_features.pkl
3. Run python app.py
4. Open http://127.0.0.1:5000 and signup/login
